# Tableau Interactive Data Visualization Project

## Overview
This project was completed as part of a **Data Technician Bootcamp** and focuses on building interactive dashboards using **Tableau** to explore and communicate insights from multiple datasets.

The objective of the project was to transform raw data into clear, engaging visualisations that support **data storytelling and decision-making**. The dashboards allow users to explore patterns, trends, and geographic insights through interactive visual elements.

---

## Key Skills Demonstrated

### Interactive Dashboard Design
- Built **interactive dashboards** to present data insights clearly.
- Combined multiple visualisations within a single dashboard layout.
- Enabled users to explore the data dynamically through dashboard controls.

---

### Data Exploration and Transformation
- Created **calculated fields** to derive additional insights from the dataset.
- Applied **filters and slicers** to allow users to explore specific segments of the data.
- Structured data to support meaningful visual analysis.

---

### Data Visualisation Techniques
Developed a range of visualisations to communicate patterns and trends effectively:

- **Bar charts** to compare values across categories
- **Line charts** to display trends over time
- **Pie charts** to show proportional distributions
- **Geographic maps** to visualise location-based data and regional patterns

These visualisations help transform raw data into accessible insights for users.

---

## Data Storytelling
A key focus of this project was **data storytelling**—using visualisations to guide users through the data and highlight important findings.

Dashboards were designed to:
- Highlight key trends and patterns
- Provide interactive exploration of the data
- Present insights in a clear and intuitive way

---

## Tools Used
- **Tableau**
- Interactive dashboards
- Filters and slicers
- Calculated fields
- Multiple chart types (Bar, Line, Pie, Map)

---

## Project Context
This work was completed during a **Data Technician Bootcamp**, where Tableau was used to develop practical skills in **data visualisation, dashboard design, and communicating insights through data**.
